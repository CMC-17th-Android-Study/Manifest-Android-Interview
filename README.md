## Rules

1. **캠은 필수**  
   "나 누구랑 얘기하고 있음?"

2. **“바보 같은 질문”은 존재하지 않음**  
   질문은 성장의 씨앗입니다. 언제든지 질문하세요.

3. **지각 또는 부득이한 불참은 최대 1회 허용**  
   1회를 초과할 경우 벌금 부과 (벌금 액수는 추후 논의)

## THE Way

1. 주차별로 `week1/홍길동` 형식의 브랜치를 생성하여 `.md` 파일로 본인의 학습 내용을 정리합니다.
2. 정리된 내용을 PR로 올린 후, merge하여 메인 브랜치의 개인 폴더(모듈)에 누적합니다.
